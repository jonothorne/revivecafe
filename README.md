# Revive Cafe Website

Official website for Revive Cafe - Your local cafe in the heart of the Norwich community.

## About

Revive Cafe is a community-focused cafe run by Alive Church in Norwich. All proceeds support local community work.

## Features

- Responsive design for mobile, tablet, and desktop
- Modern Montserrat typography with custom branding
- Integrated contact form (Web3Forms)
- Menu showcase
- Opening hours and location information
- Social media integration

## Location

## Local Development

To run locally:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Web3Forms for contact form submissions
- Google Fonts (Montserrat)
